В репозитории содержатся файлы конфигурации ipset для обхода блокировок роскомнадзора.

Для сокращения размера таблицы ip-адресов подсети содержащие более 20 хостов объединяются в одну запись. Этого достаточно для работы на роутере с 200 Мб памяти.
На роутере необходимо создать соответствующие сеты, скрипт для загрузки данных и правила маршрутизации, использующие эти списки.

Из списка разблокируемых хостов удалены по маскам гамблинг ресурсы.
